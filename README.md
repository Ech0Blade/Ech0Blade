# Hakkımda

FiveM sunucuları geliştirme odaklı çalışan, siber güvenlik ve oyun sistemleriyle ilgilenen bir geliştiriciyim.
Sunucu altyapıları, script sistemleri ve güvenlik mekanikleri üzerine projeler geliştirerek kendimi sürekli ilerletmeyi amaçlıyorum.

Projelerimde performans, stabilite ve güvenlik önceliklidir.
Hedefim deneyerek, araştırarak ve üreterek **en iyi seviyeye ulaşmak**.
